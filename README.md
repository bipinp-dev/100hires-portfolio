# 100Hires Portfolio Project

This repository documents the completion of the initial setup task for the 100Hires application process.

## Tools Installed

- Cursor IDE
- Claude Code extension for Cursor (Anthropic)
- Codex extension for Cursor
- Git
- GitHub

## Steps Completed

1. Installed Cursor IDE.
2. Installed the official Claude Code extension in Cursor.
3. Installed the Codex extension in Cursor.
4. Created a public GitHub repository.
5. Connected the local project folder to the GitHub repository using Git.
6. Created this README.md file to document the setup process.
